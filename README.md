# PuppeteerFramework
UI Automation Framework Using Puppeteer and Mocha

## Libraries used:
  - Mocha
  - Puppeteer
  
## This Simple UI Automation Framework uses:
  - Page Object Pattern
  - Screenshots for Errors
  - Love <3

## Instructions
  - `npm install` to install the modules needed
  - `npm run test` to run the test challenge
